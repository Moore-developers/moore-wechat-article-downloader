# Moore 微信公众号文章下载器

用自然语言下载、同步、整理微信公众号文章。输出 Markdown、图片和 `index.csv`，方便后续处理。

---

## 直接下载文章

有 URL 就能用，无需登录。

```text
下载这篇：https://mp.weixin.qq.com/s/xxx
```

支持一次传多个 URL、`.txt`、`.csv`、`.json` 文件。重复运行会跳过已成功的链接。

输出到 `~/Downloads/wechat-articles/<公众号名>/`：

```
articles/<标题>.md
images/<标题>/
index.csv
```

## 获取公众号历史文章

根据是否愿意登录，有两种方式。

### 扫码登录（推荐）

不需要拥有目标公众号，扫码登录自己的公众号后台即可，auth-key 有效期约 4 天。

```text
列出「某公众号」最近 50 篇让我选
下载「某公众号」最近 3 篇
```

抓取完成后会在对话中列出标题和日期，由你选择后再下载，不会自动全量下载。

### 不登录，用代理

```text
我不登录，用代理模式获取这个公众号的历史文章：https://mp.weixin.qq.com/s/xxx
```

需要在微信桌面客户端内置浏览器里打开旧版历史入口并向下滚动，只能获取实际加载出来的文章。

## 抓取评论和互动数据

普通下载拿不到评论、阅读数、点赞数，需要开启代理增强后手动触发保存。

### 开启

```text
开启代理增强
```

首次使用需要安装并信任 mitmproxy 证书（按提示操作）。

### 在微信里保存文章

开启后，在微信桌面客户端打开文章，页面加载完成后点击注入的按钮：

```text
保存这篇
```

可以获取：

- 页面已加载的评论（结构化为昵称、地区/时间、点赞、回复、内容）
- 阅读数、点赞数、评论数、在看数
- 正文 DOM、互动区域 DOM、图片 URL、页面排版风格

无法保证：全量评论、收藏数、页面未显示的数据。

### 整理保存记录

连续保存多篇后：

```text
已保存，帮我整理
```

批量处理所有未整理的记录：已下载的文章补充互动数据，未下载的从快照生成 Markdown。互动数据写入文章 Markdown 的 `## 页面数据` 区块，原始 JSON 保留在 `snapshots/`。

### 关闭

```text
取消代理增强
```

恢复系统代理，不删除已保存的文章。如需完全停止服务，说"停止代理增强服务"。

## 订阅多个公众号并定时同步

长期跟踪多个公众号，自动增量同步并下载新文章。

```text
订阅「某公众号」
同步所有订阅的公众号
下载所有新文章
```

也可以设置定时任务，每天自动执行。

---

## 安全边界

- 不绕过登录、付费墙或私密内容
- 不记录 cookie、auth-key、token、pass_ticket 等敏感凭据
- 代理增强只保存脱敏后的页面信息
- 页面未暴露的数据标记为 `missing`，不猜测
